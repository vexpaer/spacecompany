# 概述:
太空公司是一个科幻小说增量游戏，您可以从地球上的谦逊的起点开始，在星系之间旅行，搭建戴森星球（Dyson Spheres）以及几乎整个星系的殖民地(ies?)

目前，游戏是Beta版，这意味着游戏中经常有积极的发展，现有功能的大多数bug都将被修复。你不必担心你的储存被擦掉。不过，偶尔会有bug，我会尽力修改。
